The purpose of the project is to created an inverted facebook. Facebook is primarily used for people to be able to message and play games. However, our site focuses on the opposite where you can play games FIRST and message later!



You will be able to play a round of minigames. There will be several games that can be played, each game will have a high score displayed. Every time a new person gets a higher score the score will be replaced and user will be registered.

People would want to use the webapp to play with games and connect with people about the game.



For our Database Design we will use SQL to create a database and store all of the information we receive into this database. SQL should be a large enough database to contain all of the information
